Command-line interface¶

The easiest way to use pdb_prot_align is typically via the command-line executable that will be installed with the package. See below for usage.

Usage¶

Align proteins to reference and PDB.

usage: pdb_prot_align [-h] [-v] --protsfile PROTSFILE --refprot_regex
                      REFPROT_REGEX --pdbfile PDBFILE --chain_ids CHAIN_IDS
                      [CHAIN_IDS ...] --alignment ALIGNMENT --csv CSV
                      [--chain_identity {union,intersection,require_same}]
                      [--ignore_gaps IGNORE_GAPS] [--drop_pdb DROP_PDB]

Named Arguments¶

-v, --version

show program’s version number and exit

--protsfile

input FASTA file of protein sequences

--refprot_regex

regex for reference protein header in protsfile

--pdbfile

input PDB file

--chain_ids

chains in PDB file, must be identical monomers if > 1

--alignment

created FASTA alignment of proteins with gaps relative to reference are stripped; non-stripped alignment with PDB protein written to file with suffix “_unstripped”

--csv

created CSV with sequential sites in reference, PDB sites, wildtype in reference, wildtype in PDB, amino amino acid, frequency of amino acid, site entropy in bits, number of effective amino acids at site

--chain_identity

Possible choices: union, intersection, require_same

if sites differ among chains, get union, intersection, or raise error if not same sites

Default: “union”

--ignore_gaps

ignore gaps (-) when calculating frequencies, number effective amino acids, entropy

Default: True

--drop_pdb

drop protein in pdbfile from alignment and csv stats

Default: True

pdb_prot_align

Align proteins to PDB chains
